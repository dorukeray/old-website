About Web 3.0

I believe the web is one of the greatest inventions of human civilization. But you know, when we just want to get a small piece of information, all we find is useless, long web documents that are optimized for SEO —not for human— and full of privacy-infringing ads and false or unsourced information.

This big problem can be overcome with web 3.0. However, world needs a pioneer to act as a catalyst for the transition from the legacy document web to it's next phase, which will be the web of data. We want Dorkodu to be this platform. That's why we're totally focused on web technologies —like web apps, APIs, protocols, etc.

Some of My Open Source Projects

  • Sage : Query-based data exchange & API design approach to simplify the communication for data interactions of client-server apps.

  • Superbook : Awesome, simple and lightweight PHP web pico-framework. The structure of applications vary, so what if we provide single components instead of dictating a certain architecture?

  • Seekr : Minimalist testing library for writing better tests easily and wisely on PHP.

  • Lucid : Simple but productive, efficient and lightweight JS library and approach for developing component-based, awesome web apps.

    This will be an ecosystem, including many libraries & add-ons.

  • Hindsight : Markdown-based static website generator. Hindsight makes creating, deploying and maintaining static websites easier.
